Ruby - String

Normalmente criadas utilizando " ou '

        
  "Olá pessoal"
  "Olá pessoal".reverse # => "laossep álO"
  "Olá pessoal".split(' ') # => ["Olá", "pessoal"]
  "Olá pessoal".gsub('Olá', 'Adeus') # => "Adeus pessoal"

Ruby - Integer/Float

Representam números inteiros, ou decimais como 150 ou 9.99

        
        10 + 5 # => 15
        6.next # => 7
        1.3.round # => 1

Ruby - Array

São coleções ordenadas com qualquer tipo de objetos, normalmente criadas com Array.new ou []

        
[15, "Hello", "Ruby"]
[15, "Hello", "Ruby"].first # => 15
["Ruby"].push("Rails") # => ["Ruby", "Rails"]

Ruby - Symbol

Símbolos representam nomes dentro do interpretador, criados com :, ex: :coelho

        
        :caiena
        :"caiena"

Ruby - Hash

São dicionários, contendo chaves únicas e valores pode ser criado com Hash.new ou de forma implícita {}

        
notas = {}
notas['Ana'] = 10
puts notas # => { "Ana" => 10 }

Ruby - Blocos

Em ruby tudo é objeto (menos blocos)

Trecho de código fechadas nas keywords do/end ou entre {}

Um bloco pode possuír zero ou mais argumentos, definidos entre ||

        
         ['a', 'b', 'c'].each do |letter|
           puts letter
         end
